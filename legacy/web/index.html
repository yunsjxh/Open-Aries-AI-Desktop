常见问题

FAQ

常见问题解答，帮助你快速解决问题

程序启动后闪退怎么办？

检查是否安装了必要的运行库，或查看 aries_agent.log 日志文件获取详细错误信息。确保你的系统满足 Windows 10/11 的要求，并且显卡支持 GDI+。

API Key 验证失败怎么解决？

首先检查网络连接是否正常。如果使用的是智谱 AI，请确保 API Key 正确且有足够的额度。也可以尝试使用 provider 命令切换到其他 API 提供商，如 SiliconFlow 等 OpenAI 兼容的服务。

支持哪些 API 提供商？

默认支持智谱 AI（GLM-4.6v-flash），同时也支持任何 OpenAI 兼容的 API。你可以通过 provider 命令添加自定义提供商，如 SiliconFlow、OpenRouter 等。支持保存多个提供商配置，随时切换使用。

点击坐标不准确怎么办？

程序使用相对坐标 (0-1000) 自动适配屏幕分辨率，通常能够准确定位。如果遇到坐标偏差，可能是由于 DPI 缩放或特殊显示设置导致的。可以尝试调整系统 DPI 设置，或在提示词中明确指定精确的坐标位置。

文件读取失败是什么原因？

程序会显示具体的错误原因，包括：文件不存在、无权限访问、文件被占用等。AI 会根据错误信息自动调整策略，如尝试其他路径、请求管理员权限等。确保文件路径正确，且程序有足够的权限访问目标文件。

无法启动应用怎么办？

某些 UWP 应用可能需要特殊处理，程序会自动尝试多种启动方式。确保应用已正确安装，且程序有足够的权限。你也可以尝试使用 Execute 命令直接启动应用的可执行文件。

遇到"访问量过大"错误怎么办？

程序会提示是否重试，选择 y 等待 1 秒后自动重试。如果频繁遇到此问题，建议切换到其他 API 提供商，或在非高峰时段使用。

中文输入不成功怎么办？

程序已支持中文输入，请确保使用的是最新版本。如果仍有问题，检查系统输入法设置，或尝试使用英文输入法后通过 Type 动作输入中文。

视觉模型转述失败怎么办？

程序会自动回退到控件列表模式，并记录详细错误信息到日志。检查视觉模型的 API Key 是否有效，或尝试切换到其他视觉模型。也可以直接使用 UI Automation 控件列表模式。

如何更新到最新版本？

程序内置了更新检查功能，启动时会自动检查是否有新版本。你也可以输入 update 命令手动检查更新，或关注 GitHub 仓库的 Releases 页面下载最新版本。更新时只需替换 exe 文件即可，配置信息会自动保留。
